Cognitive & Emotional Performance Scoring System

Complete Implementation Guide for Technical and Emotional Performance Calculation

System Overview

This system combines Technical Performance (Understanding, Debugging, Completion) with Emotional/Affective Dynamics (Likert-scale Before/After responses) to provide a comprehensive performance score.

Technical

Bloom's Taxonomy-weighted metrics measuring cognitive performance

Emotional

Likert-scale assessment of emotional engagement before/after

Combined

Integrated performance score for holistic assessment

Key Components

  • 3 Knowledge Domains: Understanding (60%), Debugging (20%), Completion (20%)
  • Raw Scoring: Correct (+1.0), Wrong (-0.25), Not Attempted (0.0)
  • T-Score Scaling: Transform [-0.25, 1] to [0, 1] range
  • Emotion Assessment: 7 Likert questions (1-5 scale), Before & After
  • Final Score Range: [0, 0.75] (0.5 technical + 0.25 emotional)

Technical Score Calculation

Input Data Structure

For each of 3 domains (Understanding, Debugging, Completion):

Columns needed: {Domain}_Correct, {Domain}_Wrong, {Domain}_NotAttempted

Step 1: Calculate Raw Scores

Formula:
Raw_Score = (Correct × 1.0) + (Wrong × -0.25) + (NotAttempted × 0.0)
Example: (3 × 1.0) + (1 × -0.25) + (1 × 0.0) = 2.75

Step 2: Normalize by Question Count

Formula:
Domain_S = Raw_Score / 5.0
Example: 2.75 / 5.0 = 0.55

Step 3: Apply Bloom's Taxonomy Weights

Weights (Research-based):
Understanding: 60% (foundational knowledge)
Debugging: 20% (problem-solving skill)
Completion: 20% (task execution)

Step 4: Scale to 0-1 Range (T-Score)

Formula:
T_Score = (Domain_Value + 0.25) / 1.25
T_Score = Clip(T_Score, 0.0, 1.0) // Ensure within bounds

Step 5-7: Comprehensive Learning Level (CL1)

Formula:
CL1 = T1 + T2 + T3 (can range 0 to 3)
CL1_Div3 = CL1 / 3.0 (normalized to 0-1)
CL1_Div3_Scaled = CL1_Div3 × 0.5 (final technical score, 0-0.5)

Emotional/Affective Performance Calculation

Input Data Structure

7 Likert-scale questions measured at 2 time points:

Columns needed: P1_Before, P1_After, P2_Before, P2_After, ..., P7_Before, P7_After
Response Scale: 1 = Strongly Disagree, 5 = Strongly Agree

Step 1: Convert to Decimal

Formula:
P{i}_Decimal = Response / 5.0
Example: 4 / 5.0 = 0.8

Step 2-3: Calculate Totals & Averages

Formula:
Total_Decimal = Sum(P1_Decimal ... P7_Decimal)
Average (B1/A1) = Total_Decimal / 7.0

Step 4-6: Scale & Combine

Formula:
B1_Scaled = B1 × 0.5
A1_Scaled = A1 × 0.5
Emotion_Value = (B1_Scaled + A1_Scaled) / 2.0
Emotion_Value_Scaled = Emotion_Value × 0.5 (final emotion score, 0-0.25)

Combined Performance Score

Integration Formula

Combined Performance:
Combined_Performance = CL1_Div3_Scaled + Emotion_Value_Scaled

Score Range & Interpretation

Component Input Range Output Range Weight
Technical (CL1_Div3_Scaled) [0, 1] [0, 0.5] 66.7%
Emotional (Emotion_Value_Scaled) [0.2, 5] [0, 0.25] 33.3%
Combined - [0, 0.75] 100%

Performance Levels

  • High Performance: 0.50 - 0.75 (67% - 100%)
  • Moderate Performance: 0.25 - 0.50 (33% - 67%)
  • Low Performance: 0.00 - 0.25 (0% - 33%)

Complete Example Walkthrough

Student Response Data

Understanding: 3 Correct, 1 Wrong, 1 Not Attempted
Debugging: 4 Correct, 0 Wrong, 1 Not Attempted
Completion: 5 Correct, 0 Wrong, 0 Not Attempted

Emotion Before: [5, 4, 4, 3, 4, 4, 5] → Average = 4.14
Emotion After: [5, 5, 4, 4, 5, 4, 5] → Average = 4.57

Step-by-Step Calculation

Technical Score Calculation:

  • Understanding: Raw=2.75 → S=0.55 → S1=0.33 → T1=0.464
  • Debugging: Raw=4.0 → S=0.8 → S1=0.16 → T2=0.688
  • Completion: Raw=5.0 → S=1.0 → S1=0.2 → T3=0.8
  • CL1: 0.464 + 0.688 + 0.8 = 1.952
  • CL1_Div3: 1.952 / 3 = 0.651
  • CL1_Div3_Scaled: 0.651 × 0.5 = 0.326

Emotional Score Calculation:

  • Before Avg: 4.14 / 5 = 0.828 → B1 = 0.118 → B1_Scaled = 0.059
  • After Avg: 4.57 / 5 = 0.914 → A1 = 0.131 → A1_Scaled = 0.065
  • Emotion_Value: (0.059 + 0.065) / 2 = 0.062
  • Emotion_Value_Scaled: 0.062 × 0.5 = 0.031

Combined Performance:

  • Final Score: 0.326 + 0.031 = 0.357 / 0.75
  • Percentage: (0.357 / 0.75) × 100 = 47.6%
  • Interpretation: ✓ Moderate-to-High Performance

PerformanceCalculator Class

  'debugging': {'correct': 4, 'wrong': 0, 'not_attempted': 1},
  'completion': {'correct': 5, 'wrong': 0, 'not_attempted': 0},
  'emotion_before': [5, 4, 4, 3, 4, 4, 5],
  'emotion_after': [5, 5, 4, 4, 5, 4, 5]

# Calculate scores
scores = calc.calculate(student_data)

# Display results
print(calc.format_results(scores))
